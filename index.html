Query Selector

Query itu maksudnya 'mengambil'. kalau di mysql, query itu mengambil data menggunakan command mysql
nah kalo di css itu untuk mengambil atau memilih elemen html yang mau di styling. Selector di CSS bisa dilakukan
dengan mengambil nama tag nya, nama class, atau nama id dari elemen html.
satu elemen bisa punya banyak class untuk kebutuhan tertentu. 

body {
    background-color: ...;
}

h1 {
    width: ...;
}

.deskripsi {
    background-color: ...;
}

.tombol-kirim {
    width: ...;
}

#judul {
    background-color: ...;
}

#gambar {
    width: ...;
}
mengambil elemen berdasarkan nama tag bisa dilakukan dengan menuliskan nama tag-nya
jika menggunakan class, bisa diambil menuliskan dengan titik (.) sebelum nama class-nya.
untuk mengambil elemen yang memiliki id, bisa diambil dengan menuliskan tagar (#) sebelum nama id-nya

Margin

Properti CSS margin digunakan untuk membuat ruang atau jarak diluar elemen.



properti margin punya 4 value, margin: 0 0 0 0; margin: atas bawah kiri kanan;

margin: 0 0 0 0; empat nilai ini maksudnya adalah nilai jarak di sisi atas, kanan, bawah, kiri. (searah jarum jam)

margin: 0 0; kalau hanya mengisi 2 nilai seperti ini, nilai pertama untuk atas dan bawah, nilai kedua untuk kanan dan kiri

margin: 0; kalau hanya mengisi 1 nilai artinya sisi atas bawah kiri kanan nilainya sama.


lebih baik kita praktikkan saja ya, coba isi nilai margin dibawah. agar tau margin itu untuk apaa

.kotak1 {
    margin: ;
}
.kotak2 {
    margin: ;
}
.kotak3 {
    margin: ;
}
.kotak1
.kotak2
.kotak3

Padding

Padding digunakan untuk membuat jarak didalam elemen,
sehingga komponen didalamnya akan memiliki jarak di setiap sisinya secara serentak

padding dipakai di elemen pembungkus nya, bertujuan untuk memberi space terhadap elemen didalam pembungkus

padding: 0 0 0 0; mempunyai 4 nilai, sama seperti margin, merepresentasikan atas kanan bawah kiri.


mari kita praktikkan untuk penggunaan padding,


.gambar {
    background-color: yellow;
    width: 100px;
    height: 100px;
}
.kotak-informasi {
    background-color: salmon;
    padding: ;
}




Contoh isi seperti ini:
    padding: 20px;
    padding: 15px 0 0 30px;
    padding: 40px 10px 15px 10px ;
